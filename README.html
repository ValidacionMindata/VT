Se trata de un listado de cursos y su fecha de inicio donde los controles (filtros y paginado no funcionan).

Solo es necesario modificar las clases java correspondientes

Objetivo

El objetivo es modificar la aplicación para que:
  1. Devuelva el listado de cursos ordenado por fecha ascendente y luego por nombre
    B 02/03/2019
    C 02/03/2019
    A 03/03/2019

    TIP: el método java.util.List.sort se puede definir un Comparator (función de comparación entre dos elementos) que determina orden 

    							List<String> list = Arrays.asList("C","A","B");
							list.sort((e1,e2)->e1.compareTo(e2));
							System.out.println(list); // [A, B, C]

    El Comparator espera que se devuelva

    • 1 si e1 > e2
    • 0 si e1 = e2
    • -1 si e1 < e2
  2. Implementar la capacidad de filtrado
    1. Por nombre, sin distinguir mayúsculas y minúsculas
    2. Por fecha desde, o sea todos los elementos con fecha de inicio mayor o igual

    TIP: la clase java.util.Date tiene un método compareTo
  3. Implementar paginado de forma de traer solo 10 resultados por vez, dependiendo del número de página que se elija a través de los links al final del listado. Si el parámetro page no está definido, traer los 10 primeros

    TIP: el método java.util.List.sublist(from, to) puede usarse para devolver un subarray
